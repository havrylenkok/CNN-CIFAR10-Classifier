About this project

Picture was genereted by TensorBoard based on logs from learning.

This classifier uses convolutional neural network trained on 10 classes from CIFAR 10 data set. Project was built as a part of the course work at KSPU university. You can read it here (ukrainian).


Convolutional Neural network uses 10 layer architecture. Starting from input, then Conv, MaxPool, Conv, Conv, MaxPool, FullyConn, Dropout, FullyConn, Crossentropy and output. You can see accuracy on training set on the illustration.


If you need yo use server behind this project for your needs to classify image, then you can use API. Just POST image and options to /upload. All the source code, including neural network built with TFlearn, server built with Flask and front-end built with pure JS you can find, view and use on github.
